#web_cache

web_cache or wcache, is a tool that can help you store requested resources with redis

Table of Contents
=================

  * [Installation](#installation)
  * [Usage](#usage)
  * [Architecture](#architecture)
  * [Processing flow](#processing-flow)

## Installation
`$ pip install wcache`

## Usage
- default
  `$ wcache`
- custom
  `$ wcache [--ip IP] [--port PORT]`
